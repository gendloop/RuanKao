## 进程管理

1. 在操作系统进程管理中, 若进程P运行的时间片到了, 则意味着()
    * P将进入等待队列
    * P的优先级变为最高
    * P将进入就绪队列
    * P从等待队列中被唤醒

    C

    ```mermaid
    flowchart
    运行 --等待某个事件--> 等待
    等待 --等待事件发生--> 就绪
    就绪 --调度--> 运行
    运行 --时间片到--> 就绪
    ```

2. 假设系统有n(n $\ge$ 5)个进程共享资源R, 且资源R的可用数为5. 若采用PV操作, 则相应的信号量S的取值范围应为()
    * -1~n-1
    * -5~5
    * -(n-1)~1
    * -(n-5)~5

    D

    * PV操作用于解决操作系弘中进程同步与互斥问题, 通过操作信号量S实现
    * 信号量S, 通常是一个整数, 表示可用资源的数量
    * P操作, 尝试占用一个资源
      * 动作: S = S - 1
      * 判断: 减完后 S >= 0, 资源占用成功
      * 阻塞: 减完后 S < 0, 无可用资源, 当前进程进入阻塞队列
    * V操作, 释放一个占用的资源
      * 动作: S = S + 1
      * 判断: S > 0, 资源释放成功
      * 唤醒: 加完后 S <= 0, 说明等待队列里还有被阻塞的进程, 从阻塞队列中唤醒一个去执行
    * 本题中S初始值为5, 当第1个进程申请时S=4=5-n, ..., 第n个时S=5-n=-(n-5)

    * I/O完成
