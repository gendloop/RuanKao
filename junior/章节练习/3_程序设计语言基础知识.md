## 程序设计语言概述

1. 函数main(),f()的定义如下所示. 调用函数f()时, 采用引用调用方式(call by reference), 从函数f()返回后, main()中x的值为()

    ```c++
    main()           f(int&a)
    int x = 5;       int x = 2;
    f(x);            a = x - 1;
    print(x);        return;
    ```

    * 1
    * 2
    * 3
    * 4

    A

2. 对于中缀形式的算术表达式a+b*c-d, 其后缀式为()
    * `a b c d + * -`
    * `a b + c * d -`
    * `a b c * + d -`
    * `a b c + d * -`

    C

    * 常见的算术表达式为运算符在两操作数中间, 即中缀表达式
    * 后缀表达式为栈的运用, 没有优先级判断, 没有括号处理, 便于计算机的计算
    * 中缀改后缀的方法为: 按运算符优先级加括号, 把运算符移到括号后面, 再去掉括号
    * 运算演示: `(A+B)*C => ((A+B)*C) => ((AB+)*C) => ((AB+)C*) => AB+C*`
    * 常见中缀,后缀对比

      |中缀|后缀|
      |:--|:--|
      | `A + B` | `A B +` |
      | `A + B * C` | `A B C * +` |
      | `(A + B) * C` | `A B + C *` |
      | `A * (B + C)` | `A B C + *` |
      | `A + B * C - D / E` | `A B C * + D E / -` |
